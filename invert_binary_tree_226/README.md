# 226. Invert Binary Tree

https://leetcode.com/problems/invert-binary-tree/

## Difficulty:

Easy

## Description

Invert a binary tree.

Example:
```
Input:

     4
   /   \
  2     7
 / \   / \
1   3 6   9
Output:

     4
   /   \
  7     2
 / \   / \
9   6 3   1
```
